This repository contains solutions to important Linked List problems implemented in Java.
The goal of this assignment is to strengthen understanding of data structures, pointers, and algorithmic thinking.
These problems are commonly asked in coding interviews and technical assessments.

=> Topics Covered
1> Search Element in Linked List
2> Insert Node at Given Position
3> Remove Duplicates from Sorted Linked List
4> Check Palindrome Linked List
5> Add Two Numbers Represented by Linked Lists
